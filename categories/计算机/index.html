Celery框架

什么是 Celery？

Celery 是一个基于 Python 的分布式任务队列框架，主要用于处理异步任务和定时任务。
它允许你将耗时的操作（如发送邮件、处理图像等）从主应用中分离出来，交由后台的工作进程（Worker）处理，从而提高应用的响应速度和可伸缩性。
它是一个是一个成熟的、基于消息中间件的 Python 分布式任务队列/异步任务框架。
