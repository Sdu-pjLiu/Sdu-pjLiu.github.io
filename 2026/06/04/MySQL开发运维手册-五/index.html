15.7 常用参数速查

参数 含义 查看命令
max_connections 最大连接数上限 SELECT @@max_connections;
Threads_connected 当前连接数 SHOW STATUS LIKE 'Threads_connected';
Max_used_connections 历史最高连接数 SHOW STATUS LIKE 'Max_used_connections';
long_query_time 慢查询阈值（秒） SELECT @@long_query_time;
wait_timeout 非交互连接空闲超时（秒） SELECT @@wait_timeout;
interactive_timeout 交互连接空闲超时（秒） SELECT @@interactive_timeout;
innodb_buffer_pool_size InnoDB 缓冲池大小 SELECT @@innodb_buffer_pool_size;
Slow_queries 累计慢查询次数 SHOW STATUS LIKE 'Slow_queries';
